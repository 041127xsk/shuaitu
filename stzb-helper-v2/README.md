# stzb-helper-v2

《率土之滨》PC 模拟器战报抓包分析桌面应用 — 基于 Wails + Go + Vue3 构建

## 功能

- Npcap 实时抓包采集同盟战报
- ADB 自动翻页批量采集历史战报
- 同盟成员管理
- 队伍查询与胜率分析
- 分组武勋统计
- 数据导出（Excel）
- 多数据库切换与管理

## 技术栈

- 前端: Vue 3 + Naive UI
- 后端: Go + GORM + SQLite
- 抓包: Npcap + gopacket
- 桌面壳: Wails v2

## 使用

### 系统要求
- Windows 10 / Windows 11
- 无需 Root、无需网关、无需路由器权限
- 仅限 PC 安卓模拟器环境

### 快速开始

1. 下载 Release 中的 exe 直接运行
2. 首次使用须安装 [Npcap](https://npcap.com)（普通模式即可）
3. 启动安卓模拟器（雷电、MuMu、蓝叠等），打开《率土之滨》
4. 在应用中选择或新建数据库，进入主界面
5. 点击首页「战报采集」的「开启」按钮
6. 游戏内打开同盟战报页面，数据自动抓取并入库

### 自动翻页（可选）

批量抓取历史战报：
1. 确保模拟器已开启 ADB 调试
2. 侧边栏点击「自动翻页」
3. 检查 ADB 连接状态
4. 设置滑动次数后点击「开始翻页」
5. 应用自动模拟上滑翻页，配合抓包采集历史战报

### 数据导出

在「队伍查询」页面设置筛选条件后，点击「导出 Excel」导出 xlsx 文件。

### 数据库合并

- 多份数据库可通过 `INSERT OR IGNORE` 方式合并去重
- 数据库路径可在配置文件中自定义

## 开发

```bash
# 安装前端依赖
cd frontend && npm install

# 开发模式
wails dev

# 构建
wails build
```

## 构建要求

- Go 1.23+
- Node.js 18+
- Wails CLI v2.11+
